# Exocortex — Ancient Greek Readers

22 public-domain Ancient Greek readers — adapted texts, graded stories and exercise books
(19th c.) — digitised from own scans, **OCR'd twice and collated by hand against the print**.

**Read online:** https://exocortex-ancientgreek.github.io/exocortex-greek-readers/

| | |
|---|---|
| Readers | 22 |
| Lines | 45,933 |
| Words | ~357,770 |
| Reading time | ~156 h @ 2300 wph |

Each page: line-by-line or flowing text, line numbers, light/sepia/dark themes, EN/UK interface,
deep links to any line (`#l123`), and `lang="grc"` markup so **Alpheios** works on click.

## Sources
Every reader links to its scan in the ⓘ panel. Texts are in the **public domain**;
the digitisation layer (transcription, proofreading, structure) is licensed **CC BY-NC 4.0**.
